# prometheus-query

## Prometheus HTTP Client

Asynchronous Prometheus client for the Prometheus V1 HTTP API.
The client is built using **experimental** futures-0.3 and async/await.

## Queries

The following query types are supported:

- [x] Instant
- [x] Range
- [ ] Series 
- [ ] Label names
- [ ] Label values
- [ ] Targets
- [ ] Alertmanagers
- [ ] Status
- [ ] Flags
